# techread 仕様

## コンセプト

ターミナルからQiitaの記事を読むCLIツール。

## 機能一覧

### 記事閲覧（メインモード）
- 起動すると最新N件をfzfで表示（既読・未読問わず）
- 閲覧済み記事には `✓` マークを表示
- 記事を読んだら一覧に戻る
- `q` / `Esc` で終了

### 記事表示
- rawMarkdownをターミナルに表示（rich）
- レンダリング済みページをブラウザで表示（wslview）
- 表示を切り替え可能

### 閲覧履歴
- 読んだ記事をSQLiteに保存
- 次回起動時も閲覧済みマークが維持される

### 認証
- Qiita APIトークンを環境変数から読み込む
- 未設定の場合はパブリック記事のみ閲覧可能

## 確定仕様

| 項目 | 決定内容 |
|---|---|
| 起動方法 | `techread`（単一モード） |
| 記事取得件数 | デフォルト20件 |
| 閲覧履歴の永続化 | SQLite |
| fzfプレビュー | 記事冒頭100行程度を右ペインに表示 |
| 閲覧済みマーク | fzf一覧に `✓` を表示 |

## 将来の機能拡張

  - Markdown → HTML変換（`markdown`ライブラリ等）→ tempファイル → wslviewで開く

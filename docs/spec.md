# techread 仕様

## コンセプト

ターミナルからQiitaの記事を読むCLIツール。新着・おすすめ記事の通知とLGTMイベントをポーリングで取得する。

## 機能一覧

### 通知
- 新着記事の検知
- おすすめ記事の検知
- 自分の記事へのLGTMの検知

### 記事閲覧
- fzfで記事を選択
- rawMarkdownをターミナルに表示（rich）
- レンダリング済みHTMLをブラウザで表示（wslview）
- 表示を切り替え可能

### 認証
- Qiita APIトークンを環境変数から読み込む
- 未設定の場合はパブリック記事のみ閲覧可能

## 将来の機能拡張

- ブラウザ表示をQiita URLではなく、ローカル生成HTMLで行う
  - Markdown → HTML変換（`markdown`ライブラリ等）→ tempファイル → wslviewで開く
  - 理由：Qiitaのページそのままではなく、カスタムスタイルやオフライン表示が可能になる

## 確定仕様

| 項目 | 決定内容 |
|---|---|
| ポーリング間隔 | デフォルト60秒（`TECHREAD_POLL_INTERVAL` で変更可） |
| 差分検知の永続化 | SQLite |
| 通知UI | richで色付きターミナル表示のみ |
| fzfプレビュー | 記事冒頭100行程度を右ペインに表示 |
